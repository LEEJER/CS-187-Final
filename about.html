About this webpage

This webpage was made entirely by me.

There was no copy/paste used for any of the HTML or CSS. Where the sources used were only for reference.

Notable Features:

Animation:

The buttons in the nav-bar are animated to change size when you hover and click. This is done using the "transition" property.

Smooth Scrolling:

If you click the button that links to the page you are currently on, the page will scroll down to hide the banner at the top.

Custom Scrollbar:

Notice that the scroll bar on the right is blue and round. Though, I'm not certain if this effect works on all browsers and platforms, since I have not checked.

Footer:

I have a footer at the bottom of the page. Not sure what I should add to it.

Box Shadows:

The navbar and the image on the home page both have subtle shadows. I think it looks nice and adds a bit of depth to the whole thing.

Subtle Features:

Grid:

I used the grid properties of CSS to create the two column effects on various pages with relative ease.

Custom Font:

I imported fonts from Google Fonts

Variables:

I used variables for several colors in the CSS, so that main things such as background color and font color

Text Selection:

The header text cannot be selected/highlited.

Draggability:

The images and links cannot be dragged to different locations.
